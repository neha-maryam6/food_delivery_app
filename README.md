 Food Delivery App 🍕

A complete food delivery application built with **Flutter** and **Firebase**. This app allows users to order food, while restaurants and riders can manage their respective tasks.

 Features
- **User Authentication:** Secure Sign Up and Login using Firebase Auth.
- **Role-based Dashboards:** Separate interfaces for Customers, Restaurants, and Riders.
- **Firestore Database:** Real-time data storage for orders, user profiles, and menus.
- **Interactive UI:** Smooth navigation and responsive design.

 Tech Stack
- **Framework:** Flutter
- **Backend:** Firebase (Auth, Firestore)
- **Language:** Dart

 How to Run
1. Clone this repository.
2. Run `flutter pub get` in your terminal.
3. Add your `google-services.json` file in the `android/app/` folder.
4. Connect your device and run `flutter run`.
